  • {{event_title}} - Event name
  • {{event_date}} - Event date
  • {{event_time}} - Event time
  • {{event_location}} - Event location
  • {{event_price}} - Ticket price
  • {{registration_date}} - When registered
  • {{confirmation_number}} - Unique ID
  • {{event_description}} - Event details
  • {{organizer_email}} - Contact email

  • {{number_of_participants}} - How many
  • {{special_requests}} - Special notes
